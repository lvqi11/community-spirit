# Changelog

## 0.1.0 - Open Prototype

### Added

- Static web demo for OmniWeave Community.
- POI, route, task, activity, and zh-CN i18n sample data.
- AI prompt matching for visitor guidance, package pickup, activities, repair reporting, and robot patrol.
- SVG community map with selected route highlighting.
- Property Dashboard with operational metrics and queue.
- Robot-ready JSON export for patrol workflow.
- Vite + React demo implementation.
- Shareable task deep links and current workflow JSON export.
- Data validation script and `npm run check`.
- GitHub CI and Pages deployment workflows.
- GitHub issue templates and pull request template.
- Hackathon submission package, demo script, and QA checklist.
- Windows helper scripts for static demo, Vite demo, and preflight.
- Workflow export contract and robot patrol export example.
- Competitive landscape and differentiation documentation.

### Privacy

- Public repository uses fictional or synthetic data only.
- Real community, resident, access-control, camera, and security-sensitive data are explicitly out of scope.
