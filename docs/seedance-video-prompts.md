# Seedance Prompts for Community Spirit

Use Seedance for short AI-product-launch B-roll: opening hook, pain-point transition, and outro. Keep real app screencast as the main video.

## Core Constraints

```text
Product: Community Spirit
Positioning: community life RPG + physical-AI social layer + CACP reference implementation
Tone: warm, product-focused, practical, future-facing
Data: fictional / synthetic only
Do not show: real residents, real communities, CCTV, access control, payments, robots, sensor logs, fake app UI, fake Novus dashboard
```

## Opening Hook Prompt

```text
Generate a 6-8 second 16:9 product-launch opening shot.

Scene:
A fictional community courtyard represented as a clean map-like world. A basketball court is quiet at first, then a glowing quest marker appears. A soft route line connects a residential building to the court.

Mood:
Warm, optimistic, modern civic-tech, slightly playful RPG energy, not sci-fi.

Text overlay:
"What if a quiet space became a community quest?"

Restrictions:
No real people close-ups. No real apartment name. No cameras. No access control. No payment devices. No robots.
```

## Pain-Point Transition Prompt

```text
Generate a 5-second 16:9 transition for an AI product demo.

Scene:
Abstract noisy community notices and chat bubbles fade into a calm map-based recommendation card. The feeling is: from scattered group-chat activity to one clear Community Pulse.

Text overlay:
"From noisy notices to playable participation"

Style:
Clean Chinese AI-product ad style, warm, minimal, readable, not cluttered.

Restrictions:
No real chat app logos. No real resident faces. No fake product UI pretending to be the actual app.
```

## Product Loop Transition Prompt

```text
Generate a 5-second 16:9 motion graphic.

Visual:
Five clean labels animate in sequence:
"Pulse" -> "Route" -> "Check-in" -> "Reward" -> "World Ops"
Use a subtle map-grid background, small quest markers, and soft green-blue accents.

Style:
Modern product launch, crisp captions, smooth motion, judge-friendly.

Restrictions:
No people, no robots, no cameras, no payment devices, no fake dashboard.
```

## Outro Prompt

```text
Generate a 5-second 16:9 closing card.

Text:
"Community Spirit"
"The social layer physical AI needs before everyday deployment"
"Synthetic data only"

Visual:
Fictional community map, route line, quest marker, soft World Ops pulse.

Style:
Warm, clean, product-demo polish, grounded future-tech.

Restrictions:
No real residents. No real communities. No surveillance. No payments. No robot footage.
```
