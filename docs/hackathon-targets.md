# Hackathon Target Shortlist

Updated: 2026-06-16

This shortlist ranks hackathons by fit for Community Spirit's current state:

```text
community life RPG
+ real spatial operations
+ Community Pulse
+ retention loop
+ World Ops
+ robot-ready exports
```

## Recommendation

Submit to one near-term product/general AI hackathon immediately, then prepare a more tailored agent/orchestration submission for a larger AI-agent competition.

Priority order:

1. Mind the Product: World Product Day - fastest fit, product narrative friendly.
2. Qwen Cloud AI Agent Hackathon - best near-term AI-agent track if we add a thin agent framing.
3. UiPath AgentHack - strong prize pool, but requires UiPath platform integration before submission.
4. Agentic AI Build Week 2026 - strong real-estate/physical-AI fit, but likely requires on-site/team alignment.
5. DevNetwork API + Cloud + AI Hackathon 2026 - good later-stage target with enough time to tailor.

## Tier 1: Submit Soon

### Mind the Product: World Product Day - Everyone Ships Now

Link:

```text
https://mindtheproduct.devpost.com/
```

Deadline:

```text
2026-06-20 17:00 BST
```

Fit:

- Strong product-story fit.
- Accepts online submissions.
- Focuses on shipping with AI tooling and product craft.
- Community Spirit already has a live demo, GitHub repo, release, and 93-second video.

Risk:

```text
Everyone ships now, but most shipped AI products are not grounded in real places. Community Spirit shows how an AI-assisted product can turn real community spaces into repeatable participation, benefits, and retention.
```

Action:

- Use `docs/hackathon-submission-form.md`.
- Submit with current GitHub, Pages demo, and MP4 video.
- No product changes required.

Link:

```text

```

Deadline:

```text
2026-07-09 14:00 PT
```

### UiPath AgentHack

Link:

```text
https://uipath-agenthack.devpost.com/
```

Deadline:

```text
2026-06-29 23:45 EDT
```

Fit:

- Enterprise agent orchestration theme.
- Strong match with World Ops, lifecycle, handoff, and human-in-the-loop operations.
- Large prize pool and relevant judging criteria around business impact.

Risk:

- Requires the solution to run on UiPath Automation Cloud.
- Current product does not use UiPath yet.
- More integration work than a quick submission.

Recommended track:

```text
UiPath Maestro Case
```

Recommended angle:

```text
Community Pulse as a dynamic case: recommendation -> join -> route -> check-in -> benefit -> retention -> operator follow-up.
```

Required tailoring:

- Build a minimal UiPath case/process mock or real Automation Cloud workflow.
- Show where human operator approval and automation handoff happen.
- Update README and video to mention UiPath components.
